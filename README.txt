*使用Python版本为 Python 2.7.10
1. 数据结构转换(csv_transfer.py)
    1.1 csv转换为json
        直接读取后，按换行符分割，再按行用逗号分割，得到一个二维数组。
        使用zip将二维数组按列分割，将每个位置的元素创建成一个节点对象，包含位置信息，值，子节点(初始化时为空列表)
        按列从后到前遍历，找到父节点(列数-1，行数<=自身所在行数),将子节点放入父节点的子节点列表中
        递归生成dict,转化成json
    1.2 找关键字
        递归往下搜索，用一个列表记录下路径。

2. 爬取ahmad_monk的instagram主页(monk_ins.py):
    2.1 全量爬取:
        使用: python monk_ins.py full
        爬取方式: 通过抓取网页发送的http请求拿到翻页获取媒体数据的接口，
                筛选出图片
    2.2 存储方式:
        使用SQLite作为存储方式，保存图片信息，同时作为全量爬取时的去重手段、
        增量爬取时的标记。
        图片信息包括: 图片id，发布时间戳，评论数，点赞数，图片链接，图片分辨率。

    2.3 增量爬取:
        使用: python monk_ins.py increment
        爬取方式: 由于接口获得的图片的发布时间按时间倒序，所以爬取时按照全量爬取的方式爬，
                遇到数据库中存在的id停止此次爬取。
    爬取结果也在附件中(ins.sqlite)
